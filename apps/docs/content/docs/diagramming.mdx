# Діаграми

AF Docs використовує один компонент `<Diagram />` для Mermaid, PlantUML, C4 PlantUML і Graphviz. Сторінки документації зберігають складні діаграми як файли у `apps/docs/diagrams`.

## Формати

| Формат | Використання |
| --- | --- |
| Mermaid | Sequence diagrams |
| PlantUML/C4 | C1-C4 і ERD |
| Graphviz | Додаткові flow diagrams за потреби |

## Приклад

```mdx
<Diagram lang="mermaid" path="c1-context.mmd" alt="AF context" />
```

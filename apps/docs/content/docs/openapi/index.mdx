# AF API Reference

Цей розділ генерується з FastAPI `app.openapi()` у `af_api`. Схема зберігається у `apps/docs/openapi/af-api.json`, а MDX-сторінки endpoint reference створюються командою:

```powershell
pnpm build:openapi
```

Генератор не запускає web server і не виконує lifespan startup. Він лише імпортує FastAPI app і читає OpenAPI metadata.
